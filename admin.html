🔑 Hướng dẫn đăng nhập / đặt lại mật khẩu

Tài khoản đăng nhập là Firebase Authentication.

Email mặc định: admin@vangkimbao.com


Để đặt lại mật khẩu:

1. Truy cập console.firebase.google.com

2. Chọn project vangkimbao

3. Vào Authentication → Users

4. Tìm email admin@vangkimbao.com → nhấn ⋮ → Reset Password

5. Kiểm tra hộp thư email để lấy link đặt lại mật khẩu


Nếu chưa có tài khoản, nhấn "Add user" và tạo mới với email trên.
